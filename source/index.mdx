## Introduction

React Scroll Utilities provides tools to simplify many use-cases with
scrolling parts in React, whilst leaving you full control.

**The library is currently in an experimental state and subject to breaking
API changes.**

## Features

- List virtualization
- Scroll anchoring (coming soon)

## Installation

```bash
npm install react-scroll-utilities
# or
yarn add react-scroll-utilities
```
